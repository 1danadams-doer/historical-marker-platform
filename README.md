# Historical Marker Platform

## Mission

The Historical Marker Platform exists to preserve, connect, and share the stories of people, places, events, organizations, and historical resources for future generations.

Historical markers often serve as the public entry point into local history. Behind every marker lies a larger story that may include photographs, documents, oral histories, buildings, landscapes, institutions, events, people, and community knowledge.

## Repository Structure

- `/standards` — Standards, governance documents, logical data models, and source mapping specifications.
- `/schema` — Database schemas, PostgreSQL/PostGIS scripts, migrations, and implementation artifacts.
- `/etl` — Data ingestion, transformation, validation, and loading scripts.
- `/docs` — Project documentation, planning materials, technical notes, and user-facing guides.
- `/data` — Sample datasets, reference data, test data, and source mapping examples.
